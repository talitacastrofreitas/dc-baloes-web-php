# DCBalões 🎈

O DCBalões é um sistema web desenvolvido para uma loja de balões, com foco tanto na apresentação dos produtos quanto na gestão administrativa do negócio.

A plataforma exibe diversos tipos de balões e decorações, como:
- Balões personalizados
- Arranjos
- Arcos de balões
- Bubble balloons
- Kits de festa
- Decorações temáticas

Além da vitrine de produtos, o sistema possui uma área administrativa completa, onde é possível:
- Cadastrar produtos e serviços
- Gerenciar itens da loja
- Criar orçamentos personalizados
- Enviar o orçamento diretamente ao cliente via WhatsApp

O objetivo do sistema é facilitar o atendimento ao cliente, agilizar a criação de orçamentos e organizar a gestão da loja.
